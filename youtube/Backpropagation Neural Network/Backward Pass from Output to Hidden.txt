คราวนี้นะครับ ในจังหวะ เอ่อ นะครับ ในจังหวะ เนี่ยเราก็สามารถที่จะทําเหมือนเหมือนตอนที่เราปรับเบรดของ จําได้ใช่ไหมครับ คือเรามอง เรามองแบบนี้ได้เลยนะครับในจังหวะ เนี่ย เอ่อ เราสามารถดูเฉพาะส่วนได้นะครับ ก็คือดูก้อนนี้ก่อนนะครับนะครับ ดูก้อนนี้ นะครับ ก็สามารถที่จะ เนี่ยครับ
ปรับเวท คือ ณ ตรงเนี้ยผมมั่นใจว่าพวกเราหลายคนน่าจะนึกออกว่าตอนที่เราปรับเวทของเอ่อซิกมอร์ยูนิตอ่ะนะครับ เราก็ปรับแบบนี้แหละนะครับก็คือใช้เอ่อเออเรอร์นะครับ ปรับเออเรอร์ตามเจ้า เวทนะครับ เพื่อให้สามารถดูตามสเปสของ ถ้าจําได้นะครับ เรามีเวท
นะครับ แล้วก็มี error surface ใช่ไหม แล้วเราก็มีความชันนะครับ แล้วก็ปรับเวทตามความชันใช่ไหมนะครับ ตามนั้นนะครับ เพราะฉะนั้นไอเดียก็จะคล้ายๆ กันนะครับ เราจะดูพาร์ทนี้ก่อนนะครับ นะครับเอาล่ะนะครับ เพราะฉะนั้นนะครับ ใน backward path นะครับนะครับ ก็ในจังหวะ backward นะครับนะครับ ในจังหวะ backward
ก็คล้ายๆกันกับจังหวะ forward นั่นแหละแต่ว่าเรากำลังต้องเทียบนะครับเราอยากจะหา error นะครับเป็น term ของ error นะครับที่เกิดขึ้นนะครับเป็น error ที่เกิดขึ้นใน
Layer Output นะครับ เน้นนะครับตอนนี้เป็น Error ที่เกิดขึ้นใน Layer Output นะครับของเทียบกันกับ Weight ของจาก Hidden ไปยัง Output นะครับเพราะฉะนั้นนะครับก็เหมือน ๆ ตอนที่เราทำ Sigmoid นะครับเราก็จะเทียบเป็น Chain Rules นะครับก็เอา Error เทียบกันกับ
signal ของ yj ก่อนนะครับนะครับจากนั้นคูณกันครับ dip ของ s ของ yj นะครับเทียบกันกับ yj นะครับแล้วก็สุดท้ายนะครับก็ dip yj นะครับเทียบกันกับ input ของมันนะครับก็คือ
นะครับ เวทของHJ นะครับ
ก็เป็น chain rule ปกตินะครับอันนี้ก็ไม่ได้มีปัญหาอะไรนะครับคราวนี้นะครับก็มาดูแต่ละตัวนะครับไล่ตั้งแต่ deep error นะครับเทียบกับตรงนี้เป็น option นะเป็น option นะตั้งแต่เปิดต้นคลิปมาตรงนี้คือ option นะครับไม่ได้ออกสอบนะครับแต่ว่าอธิบายว่าแนวคิดมันมายังไงเดี๋ยว
พูดถึงออกสอบก็ต้องมานั่งคิดกันเนาะใจเย็นๆนะครับผมก็ผมเองก็ยังไม่รู้เหมือนกันว่าสุดท้ายเราจะเราจะจัดสอบกันท่าไหนยังไงนะครับก็บอกเลยว่ายากจังนะครับนะครับก็อันนี้นะครับก็จะได้เป็นนะครับอย่าลืมนะครับ error error ของเรานะครับ error ตัวนี้นะครับerror ก็เป็น 1 ส่วน 2 นะครับแล้วก็เป็น target นะครับ
target-output นะครับ output ก็จริงๆก็จะเป็นในอันนี้เราก็ใช้เป็นสัญลักษณ์อันนี้ไปเลยนะครับนะครับยกกำลัง 2 นะครับเพราะฉะนั้นนะครับตรงนี้นะครับเวลาเราดริปเทียบ error นะครับก็จะดริปก้อนนี้นะครับนะครับยกกำลัง 2 นะครับ
นะครับ ก็จะได้เท่ากับนะครับ เทียบกับก้อนนี้นะครับนะครับ ก็จะได้เป็น 1 ส่วน 2 นะครับ แล้วก็ตรงนี้เป็น u กำลัง 2 เนอะนะครับ u กำลัง 2 ก็เป็น 2u du นะครับ ก็คูณกับ 2 นะครับ แล้วคูณกับ u นะครับก็ target log output นะครับ แล้วก็ du นะครับ dip นะครับ
อ่า t ลบ s y j นะครับ เทียบกับ s y j นะครับ ซึ่งตรงนี้ก็จะเท่ากับนะครับ อ่า ตรงนี้ก็หมดนะครับเป็น target ลบ output นะครับ ตรงตรงเลยนะครับแล้วก็ตรงนี้ก็จะได้ t ก็เป็นค่าคงที่นะครับ เพราะมัน dip กับศูนย์นะครับแล้วก็เหลือได้เป็นลบหนึ่งนะครับ เพราะฉะนั้นนะครับ ก็จะได้เป็นลบนะครับ
target-output ที่ได้นะครับตรงนี้ก็ไปแล้ว 1 นะครับได้เรียบร้อยนะครับส่วนก้อนที่ 2 นะครับก้อนที่ 2 คือก้อนนี้นะครับก้อนที่ 2 นะครับก้อนนี้นะครับก้อนที่ 2 ก็dip s y j นะครับเทียบกันกับ
นะครับเทียบกันกับ yj นะครับนะครับก็ deep sigmoid ครับได้แล้วครับจำอันนี้ใช้จำมาเลยนะครับตัวมันเองนะครับคูณกับ 1 ลบตัวมันเองนะครับนะครับเพราะฉะนั้นก็จะได้หน้าตาแบบนี้นะครับแล้วก็สุดท้ายนะครับ deep yj นะครับเทียบกันกับตัว w นั่นแหละนะครับh j นะครับก็จะได้เป็น
YJ ก็จะเป็น W นะครับอันนี้นะครับตัว YJ นะครับมันเป็น summation ใช่ไหมครับของ WHJ นะครับ
Drip นะครับแล้วก็คูณกับ Signalนะครับที่มาจาก Z นะครับของ H ใช่ไหมนะครับใดๆนะครับเทียบกันกับ WHJ นะครับนะครับเพราะฉะนั้นนะครับเพราะฉะนั้นตรงนี้นะครับตรงนี้นะครับไอ้จังหวะที่ทำในแต่ละลิ้งค์แต่ละลิ้งค์ตรงนี้นะครับมันจะมีแค่ตัวเดียวนะครับเพราะฉะนั้น
เราก็จะได้เป็น ดิบเทียบกันก็จะเหลือแค่นี่แหละครับนะครับเพราะฉะนั้นในแต่ละลิงก์นะครับ เราก็จะปรับเวทย์หน้าตาแบบนี้นะครับอันนี้คือเราสนใจในระดับลิงก์แล้วนะครับ เพราะฉะนั้นเราเอาอันนี้ออกได้นะครับเพราะฉะนั้นเอามารวมกัน 3 ก้อนนะครับ ก็มีก้อนไหนบ้าง
ก้อนนี้นะครับก้อนนี้แล้วก็ก้อนนี้นะครับเพราะฉะนั้นนะครับเมื่ออยากรู้ว่า Dip Error เทียบกับ WHJ ที่เข้ามาแล้วนะครับในแต่ละก้อนแต่ละก้อนนะครับจะเท่ากับนะครับก็ไล่มาตั้งแต่เทอมแรกนะครับลบนะครับ Target ลบนะครับOutput นะครับคูณกับนะครับตัวมันเอง
คูณกับ 1 รอบตัวมันเองนะครับนะครับแล้วก็คูณกับขาเข้านะครับก็คือสัญญาณที่เข้ามาจาก Hidden นะครับก็เราก็จะตั้งชื่อเทอร์มนะครับตั้งชื่อเทอร์มต่างๆนะครับว่าตรงนี้นะครับเราจะตั้งว่าตรงนี้คือตัว Error นะครับ
ตัว E นะครับเป็นตัว Error นะครับแล้วก็จะตั้งว่าตรงนี้นะครับไม่พอนะครับเอาใหม่นะครับตรงนี้นะครับจะเรียกว่าเป็น Delta นะครับเป็นตัว Delta นะครับเป็นตัว Delta ของ Node ในชั้น Output นะครับก็คือ Node J นะครับเพราะฉะนั้น
เราจะเขียนได้ว่าเป็นลบเดลต้าของโน้ต j คูณกันกับ signal function ก็คือคูณกับ input ที่เข้ามานั่นแหละนะครับอันนี้คือ dip ก้อนนี้นะครับ แล้วก็ได้แล้วนะครับอันนี้เป็นความชันนะครับ
คราวนี้เราจะมาดูในชั้นที่ 2 นะครับ เพราะว่าชั้นที่ 2 มันต่างแล้วนะครับในจังหวะ Backward Path นะครับ ตรงนี้นะครับขออนุญาตยกตรงนี้ขึ้นไปนิดนึงนะครับ แล้วก็เรียกไปเป็น
Backward นะครับ สำหรับตรงนี้คือใน Output Layer นะครับส่วน Backward Path ตรงนี้นะครับ ตรงนี้เป็น Input Layer นะครับใน Input Layer นะครับ มันจะยุ่งกว่านิดนึงนะครับ ตรงนี้นะครับเดี๋ยวผมขออนุญาต Copy อันนี้ไปใช้เลยนะครับ
นะครับ แล้วก็โยก โยกก้อนนี้นะครับ มาแปะตรงนี้นะครับคือเรากําลังสนใจพาร์ทนี้ล่ะนะครับ เอาล่ะ มาดูกันนะครับว่า เอ่อ หน้าตาของการปรับเวทจะเป็นยังไงนะครับ เพราะว่า ตรงเนี้ย ปัญหาคือเออเรอร์ครับ
Error ครับมันไม่ชัดนะครับเดี๋ยวเราตัดจบคลิปนี้ที่นี่กันตรงนี้ก่อนดีกว่านะครับแล้วเรามาดูกันต่อว่าในการปรับ Weight ของชั้น Hidden นะครับInput ที่มา Hidden เนี่ยจะปรับยังไงเพราะปัญหามันคือมันไม่มีครับมันไม่มี Error ที่ที่ชัดเจนเหมือนของ Output นะครับเดี๋ยวมาดูกันในคลิปต่อไปครับ
