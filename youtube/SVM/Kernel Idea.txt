คราวนี้นะครับ ถ้าเกิดว่าเรามีปัญหากับการจำแนกมันเพราะว่าบ่อยครั้งเราไม่สามารถที่จะทำให้มันแยกตัวอย่างออกจากกันได้นะครับเราก็ใช้ไอเดียของการแปลงให้มันอยู่ในอีก Space หนึ่งนะครับเรียกว่าการทำ Kernel
นะครับ ไอเดียของการทำ Kernel ใน SVM เนี่ยคือมันต้องคู่กันเพราะว่า SVM จะใช้เส้นหนึ่งเส้นในการแบ่งดังนั้นถ้าเกิดว่าตัวอย่างนะครับ สมมุติว่าตัวอย่างมันหน้าตาเป็นอย่างนี้นะครับดูนะครับ สมมุติว่ามันมีนี่คือ 1 2 นะครับ 1 2 นะครับ สมมุติว่า
ผมมีตัวอย่างบวกนะครับตัวอย่างบวกผมอยู่ตรงนี้นะครับเป็นอย่างงี้เลยนะครับบวกอย่างงี้นะครับแล้วก็อยู่ตรงนี้นะครับส่วนตัวอย่างลบเนี่ยอยู่ตรงนี้นะครับคือ
แยกไม่ได้แน่ๆนะครับ แยกไม่ได้แน่ๆเพราะว่ามันลักษณะเหมือนเป็น 2 เส้นอย่างนี้นะครับ อย่างนี้นะครับ ประมาณนี้นะครับทำยังไงดีนะครับ เพราะฉะนั้นแนวคิดของการทำ Kernel เนี่ยก็คือเราก็จะ Map ครับ เราเรียกว่าเป็น Map นะครับ Map
ยกตัวอย่างเช่น เราเอา x1 กับ x2 มา
เราอยากรู้จังเลยว่า เราจะ map ได้คือ map ได้หลายแบบมากนะครับกลายเป็นว่าเราจับ x1 มายกกำลัง 2 นะครับ แล้วก็เอารูท 2 มาคูณกับ x1 คูณกับ x2แล้วตรงนี้เป็น x2 ยกกำลัง 2 ก็ได้นะครับเพราะฉะนั้นจาก input ที่มีแค่ 2 มิตินะครับ ก็จะกลายเป็น
กลายเป็น input ที่มี 3 มิติขึ้นมานะครับโดยที่เป็นค่าต่างๆ แบบนี้นะครับเพราะฉะนั้นลองยกลองดูนะครับในคลาสบวกนะครับในคลาสบวกของเรานะครับจะมีตัวอย่างซึ่งประกอบด้วยนะครับเป็นมีจุดนะครับประกอบด้วยจุด 0 1 นะครับจุด 1 0 แล้วก็จุด 1 2 1 นะครับ
ส่วนตัวอย่างลบนะครับ ก็จะได้เป็นจุด 02 นะครับ แล้วก็จุด 11 นะครับ แล้วก็จุด 22 นะครับสมมติว่ามันเป็นแบบนี้นะครับ มันก็จะ map เข้าไปสู่ space อันใหม่นะครับ ก็คือจะได้เป็นนะครับ
3 มิตินะครับคราวนี้ก็คือ 0 ยกกำลัง 2 นะครับตรงนี้นะครับ0 ยกกำลัง 2 นะครับ1 ยกกำลัง 2 นะครับแล้วก็รูป 2 นะครับคูณกับ 0 และ 1 ก็คือเป็น 0 นะครับตัวที่ 2 นะครับ1 ยกกำลัง 2 นะครับเป็น 1 นะครับ0 ยกกำลัง 2 เป็น 0 นะครับตรงกลางรูป 2 คูณ 1 คูณ 0 เป็น 0 นะครับ
แล้วก็อีกตัวหนึ่งนะครับ ก็จะได้เป็น 2 × 2 เป็น 4 นะครับ1 × 1 เป็น 1 นะครับ รูป 2 คูณ 1 คูณ 2 ได้ 2
รูป 2 นะครับ อันนี้คือตัวอย่างบวกนะครับ เพราะฉะนั้นตัวอย่างลบนะครับก็จะได้เป็นอย่างนี้นะครับ 0 0 4 นะครับตัวแรกนะครับ อย่างนี้นะครับ แล้วก็เป็น 1 รูป 2 1 นะครับ แล้วก็ได้เป็น 2 โทษนะครับเป็น 4 นะครับ เป็น 4 นะครับ อันนี้ก็เป็น 4 รูป 2 นะครับ
เราก็จะได้เป็น 3 มิติออกมานะครับเป็น Class บวกแล้วก็เป็น Class ลบนะครับซึ่งไอ้เจ้าทาง Class บวกและ Class ลบนี้นะครับมันก็สามารถที่จะ
เอาไป plot นะครับแล้วก็ทำให้เส้นเนี่ยมันมีแนวโน้มนะครับที่จะทำให้จุดเนี่ยมันกระจายตัวแล้วก็เส้นหนึ่งเส้นเนี่ยสามารถตัดได้นะครับซึ่งอันนี้นะครับก็จะกลายเป็นเขาเรียกว่าการทำเครื่องแล้วซึ่งมันก็จะมีอีกหลายๆขั้นตอนหลังจากนี้นะครับแต่ว่าอันนี้เป็นหนึ่งในไอเดียนะครับที่ทำให้เราสามารถที่จะจัดการกับ
กรณีที่ไม่สามารถจำแนกตัวอย่างได้นะครับ ก็เรียกว่าการใส่ Kernel เข้าไป
